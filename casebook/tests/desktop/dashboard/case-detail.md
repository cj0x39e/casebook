## Preconditions

1. Casebook application is open
2. A project directory containing test cases has been loaded
3. At least one test case exists in the tree

## Steps

1. Click a test case in the left sidebar tree
2. Observe the detail panel on the right
3. Check the priority indicator dots
4. Switch between rendered and raw content views

## Expected Results

1. The detail panel displays the case title, priority, and metadata
2. Priority dots are lit according to the configured priority level
3. Content view toggle switches correctly between rendered and raw views
